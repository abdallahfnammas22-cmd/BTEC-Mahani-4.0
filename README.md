# BTEC Mahani v1 — Next.js

هذه هي **النسخة الحقيقية الأولى** من واجهة المشروع.

## الموجود داخلها
- Navbar حقيقي
- صفحات فعلية:
  - `/`
  - `/dashboard`
  - `/question-bank`
  - `/workshops`
  - `/search`
  - `/advisor`
  - `/login`
- بحث فعلي على بيانات محلية داخل المشروع
- بنك أسئلة أولي بمستويات:
  - سهل
  - متوسط
  - صعب
  - شديد الصعوبة
- ورش عملية أولية
- محتوى مرجعي مبني على موضوعات من كتب الهندسة وYork Vocational English

## التشغيل محليًا
```bash
npm install
npm run dev
```
